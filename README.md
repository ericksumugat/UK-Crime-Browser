# UK-Crime-Browser
A simple application that lets you browse crimes in UK using a map. 


## Architecture
- CLEAN Architecture with MVP pattern on the presentation layer.

## Technologies and Libraries
* RxJava2/RxAndroid
* Dagger 2
* Retrofit
* Google Map
